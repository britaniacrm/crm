# @britania-crm/web-componens/InputPercentage
```js
import Form from '@britania-crm/web-components/Form'
import InputPercentage from '@britania-crm/web-components/InputPercentage'
import number from '@britania-crm/forms/validators/number.validator'
```

This component is a wrapper to [`@britania-crm/web-components/TextField`](/web-components/text-field)..

## Live basic usage
```js
import { useState, useCallback } from 'react'

import InputPercentage from '@britania-crm/web-components/InputPercentage'
```
<Playground>
  {() => {
    const [value, setValue] = useState('')
    const onChange = useCallback(
      (event) => setValue(event.target.value),
      []
    )
    return (
      <InputPercentage
        detached // important to use it outside form
        value={ value }
        onChange={ onChange }
        label="Percentage"
        name="percentage"
      />
    )
  }}
</Playground>

## Usage in Form
```js
import * as Yup from 'yup'
import { useRef, useCallback } from 'react'
import Form from '@britania-crm/web-components/Form'
import InputPercentage from '@britania-crm/web-components/InputPercentage'
import number from '@britania-crm/forms/validators/number.validator'
```
<Playground>
  {() => {
    const formRef = useRef(null)
    const schemaConstructor = useCallback(
      ({ t }) => Yup.object().shape({ percentage: number({ t })(Yup.string()) }),
      []
    )
    return (
      <Form ref={ formRef } schemaConstructor={ schemaConstructor }>
        <InputPercentage label="Percentage" name="percentage" />
      </Form>
    )
  }}
</Playground>
